# SCUPLUS 错误码

| 错误码         | 含义                   |
| ------------- |:---------------------:|
| x401x         | 认证相关(用户密码错误等等) |
| x422x         | 请求字段/参数错误 |
| x500x         | 服务器错误 |
| x510x         | 数据库错误 |
| 2xxxx         | 教务处相关错误 |
| 3xxxx         | 用户相关错误 |
| 4xxxx         | 图书馆相关错误 | 

## 教务处相关 2xxxx

| 错误码         | 含义           |
| ------------- |:-------------:|
| 24011         | 登录失败，用户名或密码错误 |
| 25001         | 教务处链接失败 |

## 用户相关 3xxxx

| 错误码         | 含义            |
| ------------- |:-------------:|
| 34011         | 用户认证失败 |
| 34221         | 参数错误 |
